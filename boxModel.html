¿Que es el Box Model?

El Box Model hace referencia a como se representa el contenido una pagina web, mediante el uso de cajas rectangulares

El agoritmo define como se muestran los elementos y imagenes , estilo a travez del layout

Box Model

Cad elementos se muestra como una caja rectangular, existen 2 tipos de elementos

  1. Elementos Inline
  2. Elementos de Bloque

En caso de tener un elemento inline se tiene que definir el width y height (ancho y alto)

En caso elemento de bloque el el ancho es realtivo al contenido

Capas del Box Model

Content-Box

Hacer referencia a la parte interior del box model, en donde va el contenido

Soy un contenxt box el contenido interno

Padding-Box

Hace referencia al espacio entre el context-box y el border-box , en otras palabras el espacio entre el contenido y el border de nueva box model

Nos permite mover el contenido interno

Tengo la siguientes propiedades de padding padding-top: 20px padding-right:2px padding-bottom:10px padding-left: 5px
(padding 20px 2px 10px 5px)

Shorthand para Padding

Para ejemplo de padding box se aplico el concepto de shorthand el cual significa que si yo quiero el siguiente estilo:

  • padding-top: 20px
  • padding-right:2px
  • padding-bottom:10px
  • padding-left: 5px
Podria simplemente colocar solo padding 20px 2px 10px 5px , y colocar los valores en sentido de las manecilals del reloj., primero top , luego right, botton y al final

Border-Box

Hace referencia a la linea que rodea el contendido de mi box model

Esta propiedad rompe el flujo basico del D.O.M, y es externo a elmento

Esta caja tiene un border de 5px con borde tipo dotted

Margin-Box

Es el espacio que se encuentra afuera del borde de la caja, representa la distancia con otras cajas o elementos

Ejemplo

Yo soy un caja y me dieron un margen top y botom de 30 y un margen right y left de 40
Soy una caja copia con los mismos estilos y el mismo margen

Shorthand para Margin

Para ejemplo de margin-box se aplico el concepto de shorthand el cual significa que si yo quiero el siguiente estilo:

  • margin-top: 30px
  • margin-right:40px
  • margin-bottom:30px
  • margin-left: 40px
Podria simplement colocar solo margin 30px 40px y colocar los 30px para top y botton y 40px para left y right

Margin Negativo

Con el margin negativo podemos hacer que elemento se fue mueva hacia fuera de la pantalla de navegador por lo un parte se ocultara

Ejemplo

Colapsado Margenes Verticales

Soy la caja 1 vertical y tengo un margin-bottom: 2rem

Soy la caja 2 vertical margin-top: 4rem;

Cuando existe colapsado de margenes verticales, el mayor valor del margen aboserbe al menor, en este ejemplo la caja vertical 1 tiene un margin botton 2rem y la caja 2 tiene un margin-top de 4rem, por lo que css va decide que aplicar el margin de mayor valor osea 4rem

Centrado Perfecto Horizontal con Margen

Soy la caja 1 centrado perfecto con margin 0 auto

Soy la caja 2 con centrado perfecto margin 0 auto

Nos permite centrar un elemento de bloque de manejra perfecta usando la propiedad margin 0 auto

Uso Especifico del Selector Universal

El selector universal permite dar estilos a todos los elementos html, su uso particar es para dar la propiedad box-sizing: border-box , para que cuando usemos margin, padding no se vea afectado el tamaño del contenedor
