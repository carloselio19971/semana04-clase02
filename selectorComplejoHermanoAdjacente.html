Selector Hermano Adjacente

El selector hermano adjacente nos permite selecionar el elemento que este al mismo nivel del hermano y siguiente solo aplica a un solo elemento.

Ejemplo

Para este caso se creo una caja , en la cual se tiene como primer elemento un emcabezado, luego 4 parrafos, en este caso se le esta aplicando estilo al hermano del componente h2 que esta dentro un div caja-3
