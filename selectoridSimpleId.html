Selector de ID

El selector de clases nos permite seleccionar un id en html para poderle asignar un estilo.

Ejemplo

Para este ejemplo se creo una div con una clase llamada caja a la cual se le dio un fondo amarillo, un tamaño y ancho de 200px y alto 300, adicionalmente se desea cambiar el estilo de un parrafo especial el cual se le asigno un id parrafo especial, el cual se le aplico un estilo con un fondo azul con letras colo blanco.
