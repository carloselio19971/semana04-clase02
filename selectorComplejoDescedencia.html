Selector de Descendencia

El selector de descendias nos permite seleccionar a todos los descendientes de un elemento padre no necesariamente un hijo directo.

Ejemplo

En esta caso estamos seleccionando a todos los parrafos hijos de la clase caja y todos sus descendientes

Para este ejemlo se creo una clase caja, dentro de esta clase se creo un primer parrafo, luego se creo una elemento div el cual contiene 3 elementos , 2 elementos h3 y un elemento parrafo, luego un elemento h3 y un ultimo elmento parrafo.

En este caso el estilo se va aplicar a todos los elementos parrafo , asi no sean desciente directo por lo que tambien se pinta el parrafo nieto
